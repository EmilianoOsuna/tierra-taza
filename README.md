# ☕ Tierra & Taza

> Una experiencia web cinemática y premium para explorar el mundo del café.

![React](https://img.shields.io/badge/React-19.2-blue?style=for-the-badge&logo=react)
![Vite](https://img.shields.io/badge/Vite-8.0-646CFF?style=for-the-badge&logo=vite)
![TailwindCSS](https://img.shields.io/badge/TailwindCSS-4.3-38B2AC?style=for-the-badge&logo=tailwind-css)
![Framer Motion](https://img.shields.io/badge/Framer_Motion-12.3-black?style=for-the-badge&logo=framer)

**Tierra & Taza** es una aplicación web moderna diseñada con una estética de alta calidad, integrando animaciones fluidas, un desplazamiento suave (smooth scrolling) y componentes interactivos para brindar una experiencia de usuario elegante e inolvidable.

---

## ✨ Características Principales

- **🎬 Diseño Cinemático:** Diseños editoriales, layouts inmersivos y una estética cuidada al mínimo detalle.
- **🌊 Desplazamiento Suave (Smooth Scrolling):** Integración con `Lenis` para una navegación sedosa y profesional en toda la página.
- **✨ Animaciones Dinámicas:** Transiciones complejas, micro-animaciones y efectos visuales impulsados por `framer-motion`.
- **💎 Glassmorphism y UI Moderna:** Elementos de interfaz estilizados con efectos de cristal translúcido y formularios minimalistas.
- **📱 Totalmente Responsivo:** Una experiencia perfecta y adaptativa, tanto en dispositivos móviles como en pantallas de escritorio.
- **⚡ Rendimiento Óptimo:** Construido con Vite y React 19 para tiempos de carga ultrarrápidos y una experiencia de desarrollo ágil.

---

## 🛠️ Tecnologías Utilizadas

- **Frontend:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animaciones:** [Framer Motion](https://www.framer.com/motion/)
- **Físicas de Scroll:** [Lenis](https://lenis.studiofreight.com/)
- **Iconografía:** [Lucide React](https://lucide.dev/)

## 📂 Estructura del Proyecto

El proyecto está organizado de manera modular para facilitar su mantenimiento y escalabilidad:

```text
src/
├── assets/             # Imágenes, logos y recursos estáticos
├── components/         # Componentes reutilizables de React
│   ├── Hero.jsx        # Sección de inicio cinemática a pantalla completa
│   ├── About.jsx       # Sección informativa "Acerca de"
│   ├── Tour.jsx        # Experiencia de recorrido (timelines interactivos)
│   ├── Reservation.jsx # Formulario minimalista para reservas
│   ├── Contact.jsx     # Información de contacto y mapa
│   └── Footer.jsx      # Pie de página de la aplicación
├── App.css             # Estilos globales y utilidades personalizadas
├── App.jsx             # Componente raíz que ensambla todas las secciones
└── main.jsx            # Punto de entrada principal de React
```

## 🚀 Instalación y Uso

Sigue estos pasos para ejecutar el proyecto en tu entorno local:

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/EmilianoOsuna/tierra-taza.git
   cd Huicicila
   ```

2. **Instalar las dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Construir para producción (opcional):**
   ```bash
   npm run build
   ```
